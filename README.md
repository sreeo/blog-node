## About
This project follows simple clean/onion architectural model. 

Improvements:
1. Add unit tests.
2. Cache results to redis instead of current caching logic. 

### Setup

Setup node 20 (use nvm)

### `nvm use 20`


Run the production build (Must be built first).

### `yarn build`


Run server

### `yarn start`
